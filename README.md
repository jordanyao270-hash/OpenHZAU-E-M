## 关于我们

华中农业大学经济管理学院开源社区是面向全院师生发起的一个开放协作社群，致力于：

- 汇集优质学习资源 — 课程笔记、复习资料、数据案例
- 孵化校园开源项目 — 工具脚本、可视化应用、学习游戏
- 搭建跨年级交流平台 — 经验分享、技术互助、项目合作

我们相信，知识在流动中增值，能力在协作中生长。

## 如何参与

任何人都可以参与，不需要你是编程高手：

| 角色 | 可以做什么 |
|------|-----------|
| 内容贡献者 | 整理笔记、写教程、校审资料 |
| 开发者 | 写代码、修 Bug、做功能 |
| 设计师 | 做图标、排版文档、设计界面 |
| 体验官 | 试用项目、提反馈、给建议 |

**简单的参与方式：**

1. Star 你感兴趣的项目仓库
2. 发现问题时提 Issue 反馈
3. Fork 项目后提交 Pull Request
4. 在群里分享你的想法和资源
